1. Installation
	1.1. unpack this zip file into a directory of your choice.
	1.2. that's all. just make sure all files are in the directory.

2. usage
	2.1. P4GPC_Namechanger
		2.1.1. drag and drop the dataXXXX.bin onto the P4GPC_Namechanger.exe
		2.1.2. enter last and first name
		2.1.3. this should also take care of calling P4GPC_Savegame_Checksum_Updater.exe. If not see 2.2.

	2.2. P4GPC_Savegame_Checksum_Updater
		2.2.1. drag and drop the dataXXXX.bin (or multiple) onto the P4GPC_Savegame_Checksum_Updater.exe
